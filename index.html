Quick Tip

01

General Operating Methods

Cancellation = Esc, Switch Cheat Input Window = Ctrl + Shift + C

02

Camera Movements

Move left/right = Arrow Left/Right or A/D, Forward/backward = Room table up/down or W, Move faster = Shift + Camera Move, Zoom in/out = Z/X or +/-, Switch Map Mode = M

03

Sims Manipulation

Transition to the family's next trial = Spacebar/N, Switch to a specific Sim = Click Sim Portrait, Fix the camera to the Sim = Right-click Sim Portrait Mouse

04

Objects Placement Tools

Undo/re-do = Ctrl + Z / Ctrl + Y, Rotation = , /. (with objects selected), Deletion = Del / Backspace, Grid Ignore Placement = Alt (while holding things), Move to Slot = M, Grid On/Off = G, Select Wall Tools = B, Top-down View Transition =T, Create a room = Shift (while holding the wall)

05

Download Sim from Gallery

If you don't like the randomly made Sim, go to the Gallery! You can upload york work from your game to the gallery and also download Sim from the gallery. (Nature and personality of the Sim remain the same)

06

Download Building from Gallery

If you want a more unique house, go to the gallery! Look around the houses made by countless players and choose the perfect house for your sim. Whether you're looking for a fully furnished house or an individual room like a furniture showroom, you'll definitely find the space you want.

06

Create Sim Random

Skipping the sim makes it super fast and easy to jump into the game. You can leave it to the game to randomize the sim in Sim Making Mode.

06

Furnished Building purchase

When you start a new game, you can move your sim straight into a prebuilt house. Think of it as a kind of beginner's house! If you're not confident in architecture or want to start a life mode right away, this is the perfect house. Most neighbourhoods have at least one prebuilt house that sims can buy.

Technical Cheats

ENTERCTRL+SHIFT+C

  • Cheat Code : Help = Lists all available commands
  • Cheat Code : resetSim {Firstname} {Lastname} = Reset your Sim
  • Cheat Code : Shift + click on Sim > Modify in CAS = Allows editing of everything but name change and inherited traits
  • Cheat Code : casclockspeed {0-10} = Lower the Create a Sim clock speed (0 is pause)
  • Cheat Code : fps {on/off} = See the fps (frames per second) on the screen while playing
